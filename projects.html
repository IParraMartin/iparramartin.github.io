Projects

  • The Visibility Project

    The Visibility Project is an interactive map to locate endangered or low-resource languages (LRLs). It has a focus on natural language processing; we aim at making LRLs more visible and drag attention for their research.

    Each language is represented by a marker on the map, the bigger the size, the more speakers it has. The color represents the NLP attention it receives. Clicking on a marker will display a popup with information about the language. The popup also contains a link to the language's NLP papers, if available.
